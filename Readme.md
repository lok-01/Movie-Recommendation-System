# Movie Recommendation System

A Content-Based Movie Recommendation System built using Python, Pandas, Scikit-learn, TF-IDF Vectorization, and Cosine Similarity.

## Features

- Popularity-Based Recommendation
- Content-Based Recommendation
- TF-IDF Vectorization
- Cosine Similarity
- Movie Recommendation Function

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
